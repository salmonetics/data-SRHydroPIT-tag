Data Query

For a given calendar year, all fish (unique Tag Codes) detected at the mainstem Columbia and Snake interrogation sites in that year, for fish released into the Snake River or above Lower Granite Dam, were identified and stored in a static query (Interrogation Detail). Subsequently, a second query of PTAGIS (Complete Tag History) was performed for each species using an intersection with the stored query, returning all detections across all years for those fish. This was repeated for each calendar year of interest. The combined queries thus retrieve all fish present in the hydrosystem for a given year, but also prior and subsequent histories for these fish.

Except for sockeye, the number of data rows returned for Complete Tag History using these criteria far exceed tha maximum of the PTAGIS online system, and therefore the task was divided further by Release Subbasin, selecting as few as one at a time (and it was occasionally necessary to restrict a query to a single release site)

Processing Steps

  • Detections at Ice Harbor Dam (ICH) were classified by upstream or downstream under the assumption that the final antenna group encountered can be used to infer the directionality. If this was Full Flow Bypass, downstream travel was assumed, and the site code was renamed "ICHFFB"
